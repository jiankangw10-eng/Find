# 找不同大挑戰

手機直式版，上下兩張圖，共 5 關，每關 5 個不同之處。

## 上傳到 GitHub Pages

1. 建立新的 GitHub repository。
2. 將本資料夾內所有檔案上傳到 repository 根目錄。
3. 到 Settings → Pages。
4. Source 選 Deploy from a branch。
5. Branch 選 main，資料夾選 /root。
6. 儲存後等待 GitHub 產生網址。

## 檔案結構

- index.html
- style.css
- app.js
- levels.js
- manifest.webmanifest
- assets/level1_a.svg ... level5_b.svg

所有圖片均為原創 SVG 向量圖，檔案小、載入快。
